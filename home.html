Name
    pamrel - REST-like pastebin API

DESCRIPTION
    Paste content of files via APIs

PASTING API
    POST     http://pamrel.lu
    {
        "content": "print 'This is a paste!'",
        "fileExtension": "py",
        "theme": "wombat"
    }

EXAMPLE
    $ cat some_file.py | curl -F 'content=<-' -X POST
    
    $ pamrel --theme=wombat some_file.c
